readMe

Using XAMPP to run the .php files, ensure bookstore.php and checkout.php are both within the xampp directory of:

C:\xampp\htdocs\CS

create the .\cs directory if need be.

In Internet Explorer, enter the URL:
http://localhost:5500/CS/bookstore.php

The port is 5500 as altered in the XAMPP directory.

Run the xampp-control application, navigate to the Apache config. and change the port.

You can now select how many of each book you want to buy, and hit add to cart to confirm your order addition.
Hitting check out brings you to the checkout page which gives all final prices.
